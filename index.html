Project 1

For the first project, I used the iPortfolio Bootstrap 5 Resume Template, downloaded from BootstrapMade Iportfolio download This template was selected because it offers a clean and structured layout suitable for a personal portfolio. Built with Bootstrap 5, it is fully responsive and allowed me to focus on content customization rather than layout development.
At the start of the project, usability and navigation were prioritized. The fixed side navigation menu was retained to allow easy access to different sections of the website. A blue accent color was chosen to create a professional appearance with strong visual contrast. Sections such as Portfolio and Testimonials were removed as they were not relevant to the project requirements.
All images from the original template were replaced with personal or royalty-free images. These images were resized and optimized to reduce loading times while maintaining good visual quality across various screen sizes.
    Background image for thr home page
    Portrait Image (JPG)
Several external tools were used throughout the project. Canva was used to design the logo, application icon, and favicon. The website is hosted on GitHub Pages, which does not support PHP-based forms. To address this, the contact form was integrated with Formspree, enabling form submissions without server-side scripting.
One of the main challenges involved the Projects section. Initially, the navigation menu linked to a non-existent section, causing navigation issues. This was resolved by creating a dedicated Projects section and updating both HTML and CSS files to maintain visual consistency. This process strengthened my skills in troubleshooting, layout customization, and understanding hosting limitations.

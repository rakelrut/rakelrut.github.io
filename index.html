Project 1

For this project, I developed a personal portfolio website using the iPortfolio Bootstrap 5 Resume Template downloaded from htmltemplates.co. The template was selected because it offers a clean, professional, and structured layout that is well suited for a portfolio website.

Rather than building a website completely from scratch, the objective of this project was to customize an existing template and adapt it to meet academic requirements.

Homepage and Content Customization

The homepage content was fully customized to reflect my education and the course I am currently studying. The original placeholder text from the template was removed and replaced with relevant academic information.

What I Changed from the Original Template

  • Removed the Portfolio and Testimonials sections.
  • Restructured the site into a Projects section.
  • Created Project 1, Project 2, Project 3, and Final Project subsections.
  • Created additional sections such as Goals.
  • Replaced template images with personal or royalty-free images.
  • Customized homepage, About, and Resume content.
  • Integrated Formspree instead of the original PHP contact form.

Design Decisions and Visual Theme

A key visual feature of the template is its alternating section design. I wanted to preserve that because it improves readability and gives the page a clear visual structure.

Image Replacement and Optimization

Project 2

This project was part of a digital fabrication assignment involving a vinyl cutter and a laser cutter. The assignment required creating a parametric press-fit model.

Vinyl Sticker

I used Inkscape to prepare a vector design for vinyl cutting and transferred the final sticker using painter’s tape.

Inspiration source: Cute cats reference image

Wall-Mounted Shelf

Idea and Brainstorming

I began by designing a wall-mounted shelf in Fusion 360 using parametric modeling.

Design

My first parametric attempt was messy, so I rebuilt the design in a cleaner way with more organized sketches and parameters.

Parametric Challenges

I learned how important master sketches and thickness parameters are when designing press-fit parts.

Laser Cutting Setup Problem

I discovered that a wrong kerf value caused Fusion 360 to misread some geometry.

Decorative Lamp

New Idea

I switched from the shelf to a decorative lamp when material availability changed.

Decorative Lamp Inspiration

Lamp Design

I used Inkscape and Fusion 360 to create decorative cut-out patterns for a laser-cut acrylic lamp.

Kerf and Joint Testing

I tested different kerf values and found that the best fit came from a kerf value of 0.17 mm and cutting speed of 22%.

Cutting and Assembly

After cutting the final parts, I assembled the lamp and installed the electrical components.

Project 3

3D Printing

Project Idea

I designed a custom trash can for a compartment in my car.

Measuring the Compartment

I used photos with a measuring tape to estimate the shape and scale of the compartment.

First Prototypes

I made cardboard and 3D printed prototypes to improve the fit before the final design.

Designing the Trash Can

I recreated the compartment geometry in Fusion and added a custom lid with a swing mechanism.

Adding Text with Inkscape

I added the text “Rusl Rakelar” to the model using Inkscape and Fusion.

Preparing for 3D Printing

I exported the model as a 3MF file, added supports in the slicer, and prepared it for printing.

What I Learned

  • Measuring irregular shapes often requires multiple methods.
  • Photos can help estimate scale, but perspective distortion matters.
  • Test prints and cardboard prototypes save time and material.
  • SVG files can be used in Fusion to add text and graphics.

3D Scan Preview

I used Polycam on my smartphone to make a full-color 3D scan of my Einstein piggy bank and displayed it using Three.js.
